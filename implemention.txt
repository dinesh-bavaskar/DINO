# Employee Management System - Implementation Plan

## Project Overview

Develop a modern and responsive Employee Management System with separate Admin and Employee roles.

### Tech Stack

#### Frontend
- React.js (Vite)
- Tailwind CSS
- ShadCN UI
- React Router DOM
- Axios
- Lucide Icons

#### Backend
- Django
- Django REST Framework (DRF)
- Django Simple JWT
- SQLite Database

#### UI Theme
- Primary Color: #38BDF8 (Light Blue)
- Secondary Color: #FFFFFF (White)
- Modern Corporate Design
- Fully Responsive (Mobile, Tablet, Desktop)

---

# Phase 1: Authentication & Employee Management

## Functional Requirements

### Admin Features

- Admin Login
- Register New Employee
- View Employee List
- Logout

### Employee Features

- Login using Employee ID
- View Profile Information
- Logout

# Database Design

## Employee Table

| Field Name | Type |
|------------|------|
| id | AutoField |
| employee_id | CharField |
| full_name | CharField |
| email | EmailField |
| department | CharField |
| designation | CharField |
| password | Hashed Password |
| role | Admin/User |
| created_at | DateTime |

# API Design

## Authentication APIs

### Admin Login

```http
POST /api/auth/admin-login/
```

Request

```json
{
  "username": "admin",
  "password": "admin123"
}
```

Response

```json
{
  "access": "jwt_access_token",
  "refresh": "jwt_refresh_token"
}
```

---

### Employee Login

```http
POST /api/auth/employee-login/
```

Request

```json
{
  "employee_id": "EMP001",
  "password": "password123"
}
```

---

### Register Employee

```http
POST /api/employees/register/
```

Access

```text
Admin Only
```

---

### Employee List

```http
GET /api/employees/
```

Access

```text
Admin Only
```

---

### Employee Profile

```http
GET /api/profile/
```

Access

```text
Employee Only
```
